## Enumeration

We kick off our initial Nmap scan, specifying to use the default scripts `-sC`. Then we specifying that we want Nmap to enmuerate the identified service versions `-sV` for the identified ports. Next we specifying that we want Nmap to output verbose output of Nmap's results as it scans so we set `-vv`. For systems that take longer for Nmap to scan, we can begin further enuemration without having to wait till it finishes. Then we specify to output the results in all formats that Nmap makes available to us with the `-oA`. Although in this particular case we are only scanning once box, if we were scanning a large network, we could use the greppable output format to output all systems running a particular service, allowing us to quickly target all systems running that service. We then specify the name of the output file `sauna-namp`, for organization purposes we specify a meaningful output file name. Finally, we specify the IP address for the Sauna target, 10.10.10.175.
